Use these filenames in `public/` so the site picks them up automatically:

- `brand-safari-logo.png`
  Main transparent logo used in the header and hero.
- `brand-safari-logo.svg`
  Fallback logo if the PNG is not present.
- `brand-safari-favicon.png`
  Optional square favicon version for browser tabs.

If you replace `brand-safari-logo.png` with the real transparent brand file, the site will use it without any code changes.
